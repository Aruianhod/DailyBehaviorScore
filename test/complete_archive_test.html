归档功能完整测试

这个页面将帮助我们全面测试归档功能的工作状态。

测试步骤：

  1. 确保前端和后端服务器都在运行
  2. 点击下方的测试按钮验证各项功能
  3. 在内嵌的应用页面中测试归档功能
  4. 查看控制台输出了解详细信息

内嵌应用页面（直接测试）

手动测试说明：

在上方的内嵌应用中：

  1. 查看管理员功能页面
  2. 找到"归档功能"卡片（紫色图标 📁）
  3. 点击归档功能卡片
  4. 验证是否正确跳转到归档管理页面
  5. 检查是否显示"返回主页"按钮

调试脚本

如果归档功能无法正常工作，请打开浏览器开发者工具（F12），然后运行以下脚本：

// 在浏览器控制台中运行此脚本 console.log('=== 归档功能调试 ==='); // 1. 检查归档卡片 const cards = document.querySelectorAll('div[style*="cursor: pointer"]'); console.log('找到卡片数量:', cards.length); cards.forEach((card, index) => { if (card.textContent.includes('归档功能')) { console.log('找到归档功能卡片:', card); // 强制触发点击 card.click(); console.log('已点击归档卡片'); // 检查点击后的状态 setTimeout(() => { const content = document.body.textContent; if (content.includes('管理毕业学生数据归档')) { console.log('✓ 归档页面加载成功'); } else { console.log('✗ 归档页面未正确加载'); } }, 1000); } });
